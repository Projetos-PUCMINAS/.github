## Projetos de disciplina

| Repositório | Descrição | Linguagem principal |
|---|---|---|
| [Sistema-de-Aluguel-de-Carros](https://github.com/Projetos-PUCMINAS/Sistema-de-Aluguel-de-Carros) | Sistema web para gerenciamento de aluguel de veículos com análise de crédito. Desenvolvido em Spring Boot com PostgreSQL: clientes criam pedidos, funcionários avaliam solicitações e agentes bancários analisam crédito, com controle de acesso por roles e contratos automatizados. | Java |
| [sistema-de-matricula](https://github.com/Projetos-PUCMINAS/sistema-de-matricula) | Sistema de matrícula acadêmica: gerencia cursos, disciplinas, alunos e professores, viabilizando a geração da grade semestral, matrícula de alunos, visualização de turmas pelos docentes e notificações de cobrança. | Java |
| [Sistema-de-Gerenciamento-de-Congressos-Empresariais](https://github.com/Projetos-PUCMINAS/Sistema-de-Gerenciamento-de-Congressos-Empresariais) | Sistema de gerenciamento de congressos empresariais, com funcionalidades de cadastro, agendamento e controle de participantes. | Java |
| [sistema-de-controle-de-ceremonial-de-formatura](https://github.com/Projetos-PUCMINAS/sistema-de-controle-de-ceremonial-de-formatura) | Sistema de controle de cerimonial de formatura para uma firma especializada, com diagrama ER, modelo relacional e comandos SQL para criação de tabelas, índices e consultas. | SQL / Modelagem de dados |
| [Hotel-Descanso-Garantido](https://github.com/Projetos-PUCMINAS/Hotel-Descanso-Garantido) | Sistema de gerenciamento para o Hotel Descanso Garantido (Itacaré – BA), substituindo controles manuais por uma solução automatizada de estadias, clientes e funcionários. Projeto da disciplina de Fundamentos de Engenharia de Software e Algoritmos e Estruturas de Dados I. | C |
| [PersonalWebsite-JSONServer](https://github.com/Projetos-PUCMINAS/PersonalWebsite-JSONServer) | Site pessoal dinâmico que utiliza JSON Server para simular uma API RESTful e integra com a API do GitHub para exibir informações e projetos. | CSS / JavaScript / Node.js |
| [Oficina_SpringBoot](https://github.com/Projetos-PUCMINAS/Oficina_SpringBoot) | Materiais e exemplos da oficina de Spring Boot do professor João Aramuni, focada no desenvolvimento de aplicações com Spring Boot. | Java |
| [laboratorio-de-desenvolvimento-de-software](https://github.com/Projetos-PUCMINAS/laboratorio-de-desenvolvimento-de-software) | Repositório de apoio da disciplina de Laboratório de Desenvolvimento de Software, com links de ferramentas e recursos utilizados (IntelliJ, Spring Initializr, PrimeFaces, Mantine, Insomnia, Postman, entre outros). | HTML |
